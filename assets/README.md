# ResQ-AI Asset Dependencies

Download these packs from https://docs.omniverse.nvidia.com/usd/latest/usd_content_samples/downloadable_packs.html
and unzip them into this folder:

1. **City Demo Assets Pack** (193 MB) → `assets/CityDemo/`
2. **Rigged Characters Asset Pack** (891 MB) → `assets/Characters/`
3. **Extensions Sample Asset Pack** (159 MB) → `assets/Particles/`
4. **Base Materials Pack** (8.2 GB) → `assets/BaseMaterials/`
5. **Environments Skies Pack** (8.9 GB) → `assets/Environments/`
